How to Use It

Option 1: Terminal (CLI)
  1. Open Terminal (Mac: Cmd+Space → type "Terminal")
  2. Type: cd {{install_path}}
  3. Type: campaign-respond new ~/path/to/questionnaire.pdf
  4. Check progress: campaign-respond status
  5. Full command reference: campaign-respond help
Option 2: Claude Desktop App
  1. Download Claude desktop app from claude.ai/download
  2. Open Claude → Settings → Developer → Edit Config
  3. Add this to your claude_desktop_config.json: 
    {
  "mcpServers": {
    "campaign-respond": {
      "command": "python3",
      "args": ["{{install_path}}/mcp-server/campaign-respond-server.py"]
    }
  }
}
  4. Now you can chat naturally:
    • "Process this questionnaire for the Apple Ballot"
    • "Show me the status of my current questionnaires"
    • "Add this speech transcript as source material"
